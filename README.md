# **Wordform statistics**

Микросервис на **FastAPI** для автоматизированного анализа текстовых файлов: подсчет частотности слов, лемматизация (приведение к начальной форме) и генерация отчетов в формате Excel.

## **Основные возможности**

* **Стриминговая обработка:** Чтение файлов реализовано через BinaryIO (потоковое чтение по строкам). Это позволяет обрабатывать файлы объемом до **2 ГБ** при стабильном и низком потреблении RAM.  
* **Умная лемматизация:** Использование pymorphy3 для корректного морфологического анализа (например, *"бежал"*, *"бежим"*, *"бежать"* будут автоматически объединены в одну лемму).  
* **Excel-отчеты:** Генерация таблиц с результатами анализа с использованием Pandas и openpyxl.  
* **Построчная аналитика:** Помимо общей статистики по документу, сервис сохраняет данные о вхождении слов в каждой строке.

## **Технологический стек**

* **Язык:** Python 3.12+  
* **Framework:** FastAPI  
* **Data Science:** Pandas, Openpyxl  
* **NLP:** Pymorphy3 (Морфологический анализатор для русского языка)  
* **Package Manager:** [uv](https://docs.astral.sh/uv/)
* **DevOps:** Docker (Multi-stage build), Docker Compose

## **Как запустить**

### **1\. Быстрый запуск через Docker Compose**

```bash
docker-compose up --build
```

Документация API (Swagger) будет доступна по адресу: http://localhost:8000/docs

### **2\. Локальный запуск (через uv)**

Установка зависимостей (игнорируя dev-пакеты)
```bash
uv sync --no-dev
```

Запуск тестов:
```bash
uv run pytest
```

Запуск сервера:
```bash
uv run uvicorn src.main:app
```
